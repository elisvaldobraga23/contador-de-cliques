# 📌 Contador de Cliques

Uma aplicação simples desenvolvida com React que implementa um contador de cliques.
Cada vez que o botão é clicado, o contador é incrementado em +1.

Este projeto faz parte de um desafio prático do curso **DevStart**, com foco na prática dos conceitos fundamentais do React.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* 🟨 JavaScript
* 🎨 CSS
* 📦 Create React App

---

## 📷 Funcionalidades

✅ Exibir o valor atual do contador
✅ Incrementar o contador ao clicar no botão
✅ Atualização dinâmica da interface usando estado (`useState`)

## 💻 Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/elisvaldobraga23/contador-de-cliques.git
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd contador-de-cliques
```

### 3️⃣ Instale as dependências

```bash
npm install
```

### 4️⃣ Execute o projeto

```bash
npm start
```
